üìã Overview

This workflow ensures all systems stay synchronized when making monthly updates. The order matters because:

  • Schema Tools must generate schemas before Chat AI Bot ingestion (Chat Bot needs the schemas)
  • Schema deployment must happen before Squarespace can fetch updated schemas
  • AI GEO Audit monitors changes after everything is live

Create/Update Source CSV Files in Shared Resources
Export data from Squarespace and other sources, then update the raw CSV files in alan-shared-resources/csv/.

Data Sources & Update Process:

üìÖ Events (Reschedule Expired Events):

‚ö†Ô∏è Important: Events are composite entities made up of workshops, courses, AND products. When rescheduling events, you must update ALL related CSV files.
  • Source: Squarespace ‚Üí Export events, workshops, courses, and products
  • Files to Update (6 total):
    • Workshop CSVs (2 files):
      • 03-photographic-workshops-near-me.csv

  • File to Update: 01-blog-posts.csv
  • Action:

  • Note: raw-03a-trustpilot-reviews-historical.csv is historical/one-off - do not update

üîó Site Structure:

  • Export fresh data from Squarespace before updating CSVs
  • Keep file naming conventions (lowercase, hyphens, numbered prefixes)

Process & Clean Data (If Needed)
Run data processing tools to clean, match, and merge data. This creates files in csv processed/.

Processing Tasks:

  • Product Cleaning: Process raw product CSV ‚Üí csv processed/02-products-cleaned.xlsx
  • Review Matching: Match Trustpilot + Google reviews ‚Üí csv processed/03-combined-product-reviews.csv
  • Event-Product Mapping: Generate mappings ‚Üí csv processed/event-product-mappings-*.csv
Note: Only run if you've updated products or reviews. Blog/event updates may not require this step.

Generate Schemas (Schema Tools)
Use Schema Tools to generate JSON-LD schemas from your updated CSV data. This must happen before Chat AI Bot ingestion.

Schema Generation Order:

  • Blog Schema: Generate from 01-blog-posts.csv ‚Üí outputs/schema/blog/blog-schema.json
  • Workshop Schema: Generate from workshop CSVs ‚Üí outputs/schema/workshops/workshops-schema.json
  • Lessons Schema: Generate from lessons CSVs ‚Üí outputs/schema/lessons/lessons-schema.json
  • Product Schema: Generate from csv processed/02-products-cleaned.xlsx ‚Üí outputs/schema/products/*.json
  • Landing Pages: Generate if updated ‚Üí outputs/schema/
Open Schema Tools ‚Üí
‚ö†Ô∏è Critical: Schema generation must complete before Step 4. Chat AI Bot ingestion reads from outputs/schema/.
4
üì¶
Deploy Schemas to Production
Copy generated schemas from alan-shared-resources/outputs/schema/ to the alanranger-schema GitHub repo for Squarespace to fetch.

Deployment Steps:

  • Copy schema JSON files to alanranger-schema repository
  • Commit and push to GitHub
  • Schemas become available at https://schema.alanranger.com/*.json
  • Squarespace pages reference these via <script src="..."> tags
Note: This step ensures Squarespace can fetch the latest schemas. The Chat AI Bot will also use these schemas in the next step.
5

Ingest Data into Chat AI Bot
Run the Chat AI Bot ingestion pipeline. This reads from processed CSVs AND the generated schemas from Step 3.

Ingestion Process:

  • Reads from csv processed/ (cleaned product data, reviews, mappings)
  • Reads from outputs/schema/ (JSON-LD schemas generated in Step 3)
  • Reads from csv/06-site-urls.csv (site structure)
  • Ingests into Supabase tables for AI responses
  • Creates embeddings for semantic search
Open Chat AI Bot Ingest Pipeline ‚Üí
‚ö†Ô∏è Critical: This step requires schemas from Step 3 to be complete. The ingestion reads both CSV data AND schema files.

  • Analyze SERP behavior changes
  • Review performance metrics for blog posts updated with new meta titles/descriptions
Open AI GEO Audit Dashboard ‚Üí

‚úÖ Workflow Summary

The correct order ensures data flows properly through the system:

  1. Update source CSVs ‚Üí Single source of truth
  2. Process data ‚Üí Clean and match (if needed)
  3. Generate schemas ‚Üí Schema Tools creates JSON-LD
  4. Deploy schemas ‚Üí Make available for Squarespace
  5. Ingest to Chat Bot ‚Üí Uses processed CSVs + schemas
  6. Monitor performance ‚Üí AI GEO Audit tracks changes
Key Insight: Schema generation (Step 3) must happen before Chat AI Bot ingestion (Step 5) because the ingestion process reads from outputs/schema/ to enrich the AI's knowledge base.
